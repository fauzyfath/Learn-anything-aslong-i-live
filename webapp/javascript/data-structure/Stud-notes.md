# what is data structure
data structure is a way of orginizing and storing data in a computer so that it can be accessed and used efficiently.

- linear data structure: 
    a data structure which data elements are arranged sequentially or linearly, where each elements is attached to its previous and next adjacent elements. examples: array, stack, queue, linkedlist
- non-linear data structure:
    data structures where data elements are not placed sequintially or linearly. examples: tree and graphs.

# application of data structure
- database: 
    to store data in database, allowing for efficient retrieval and manipulation
- operating system:
    to design and implementation of operating system to manage system resources, such as memory and files
- computer graphics:
    to represent geometric shapes and other graphical elements in computer graphics applications
- artificial intelligence:
    to represent knowledge and information in artificial intelligence

# advantage of data structure
- efficiency: 
    data structure allow for efficient storage and retrieval of data, which is important in application where peformance critical
- flexibility:
    provide a flexible ways to orginize and store data, allowing for easy modification and manipulation
- reusability: 
    used in multiple programs and applications, reducing the need for redundant code
- maintainablity:
    well design data structure can make programs easier to understand, modify, and maintain overtime
